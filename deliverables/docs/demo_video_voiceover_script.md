# 学途智伴 2 分钟演示视频口播脚本

大家好，我们的作品叫 **学途智伴**，它是一个面向中学生的 AI 学习智能体。

这个系统的核心是 **Hermes 智能体引擎**。它不是简单回答问题的聊天机器人，而是可以围绕学生的学习材料持续工作：比如分析试卷和作业，识别重点题，整理错因和薄弱点，并进一步生成阶段性的学习建议和周报。

从架构上看，学途智伴由两部分组成：后端是 Hermes 智能体服务，负责分析、记录和生成；前端是 Web UI，负责让学生上传材料、确认重点题、输入备注，并查看结果。未来这个系统还可以继续扩展，比如接入微信提醒，把学习建议主动推送给学生或家长。

接下来我们看具体使用流程。

首先进入“学习成果”模块。学生可以上传一页试卷或作业扫描图。系统会识别其中的题目区域，并把题目列出来。这里我们可以看到，学生不是简单上传完就结束，而是可以进一步确认哪些题是重点题。比如这几道反比例函数相关题目，学生可以标记为重点，并补充得分、知识点、错因和备注。

这些人工确认的信息非常重要，因为它让 AI 的分析不是凭空生成，而是基于真实的学习证据。

接着看“输入备注”模块。学生也可以直接输入文字备注，比如今天哪个知识点没听懂，或者哪类题做起来不熟练。这些内容同样会进入 Hermes 的分析流程。

然后是“历史周报”模块。Hermes 会把上传材料、重点题、错因、薄弱点和文字备注结合起来，形成 learning findings，并更新学习 memory。之后系统会把这些信息整理成周报，展示当前学习状态、主要问题、优先复习方向，以及下一步建议。

最后看“学习内容”模块。这里可以保存教材或学习资料的理解结果，帮助 Hermes 对照学生当前进度，更准确地判断问题出在哪里。

整个流程从学习材料开始，到重点题确认，再到 AI 分析、memory 更新和周报展示，形成了一个完整闭环。它帮助中学生把零散的错题、作业和备注，转化成清晰的问题分析和可执行的学习建议，让复习更有方向，也让学习进步更容易被看见。
